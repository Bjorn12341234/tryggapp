Angående "Okänd utgivare"-varning:

Eftersom Trygg är ett gratis open-source projekt har vi inte köpt dyra signerings-certifikat från Microsoft eller Apple än. Din dator kan därför varna när du installerar. Detta är normalt för open source.

Windows: Klicka "Mer info" -> "Kör ändå".
macOS: Högerklicka på filen och välj "Öppna".

100% Lokal Data

Dina personuppgifter lämnar aldrig din enhet. Ingen molnserver, ingen databas hos oss. Du äger din data.

BankID-automation

Trygg automatiserar processen mot sajter som Ratsit och Merinfo. Skanna QR-koden, sen sköter roboten resten.

Öppen Källkod

Lita inte på ord. Lita på koden. Allt finns öppet på GitHub för granskning. Inga dolda bakdörrar.
